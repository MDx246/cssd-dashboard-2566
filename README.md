## เปิดใช้งาน

เปิดไฟล์ `index.html` ด้วย Chrome, Edge, Safari หรือ Firefox ได้ทันที ไม่ต้องต่ออินเทอร์เน็ตและไม่ต้องติดตั้ง dependency เพิ่ม

## GitHub Pages

เมื่อนำ repo นี้ขึ้น GitHub แล้ว สามารถเปิด GitHub Pages โดยเลือก:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`

หลังเปิด Pages แล้ว dashboard จะใช้งานได้ที่ URL ของ GitHub Pages ของ repo นี้
